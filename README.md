# JS Variables

Minimal JavaScript exercise focused on variables, primitive values, numeric operators, string interpolation, and basic string methods.

## Exercise Goal

Starting from the provided files, complete the requested tasks in each script and verify the results with DevTools and `console.log`.

## Tasks

- [01-numbers](01-numbers/script.js): numeric variables and arithmetic operators
- [02-strings](02-strings/script.js): variable declarations, primitive values, template literals, and string length
- [03-strings-bonus](03-strings-bonus/script.js): string search, substring extraction, replacement, concatenation, and uppercase conversion

## How to Run

Open one of the `index.html` files from the exercise folders in the browser and check the console output in DevTools.
